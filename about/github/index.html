xfce-live-wallpaper

Forked from Wolfyxon / xfce-live-wallpaper
Extended support

Merged with Wolfyxon / xfce-live-wallpaper at 2022-11-20

If it doesn't work for you, run kill.sh then,
run xfconf-query --channel xfce4-desktop -m on your terminal, then change the wallpaper once.
On the terminal you should see something like this set: /backdrop/screen0/monitor0/last-image, yours might be different.
Inside the wallpaper.py file, add a line after line 13 that says "xfconf-query -c xfce4-desktop -p /the/output/of/your/terminal -s "+path, replace /the/output/of/your/terminal with your own value (without the "Set: ")
After that try running xfce-live-wallpaper again, it should work.
